Java Movie Backend

Java Spring Boot ile JWT Korumalı Film API'si

Proje Özeti

Bu proje, Java programlama dili ve Spring Boot framework'ü kullanılarak geliştirilmiş kapsamlı bir film arka uç (backend) projesidir. Film verilerini yönetmek için RESTful API standartlarında tasarlanan proje, güvenliği merkeze alır.

Güvenlik ve Yapılandırma

Proje içerisinde güvenlik işlemleri için JWT (JSON Web Token) tabanlı Kimlik Doğrulama (Authentication) ve Yetkilendirme (Authorization) mekanizmaları kurgulanmıştır. Kapsamlı CRUD operasyonları başarıyla gerçekleştirilmiştir.

Stack

• Java • Spring Boot • Spring Security & JWT • RESTful API • SQL
